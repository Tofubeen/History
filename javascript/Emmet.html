유용한 Emmet을 익히자!

[1] '!': 정해진 양식으로 자동 완성

[2] 태그만 입력하는 경우

예시 1: h1

예시 2: input

[3] '>': 부모-자식 태그로 형성

예시: div>p

[4] '+': 형제 태그로 형성

예시: h1+p

[5] '(태그)#아이디명'

예시 1: h1+p#demo

예시 2: #demo

[6] '(태그).클래스명'

예시 1: h1+p.demo

예시 2: .demo

[7] '태그*숫자': 해당 태그의 반복

예시: p*5

[8] '태그{텍스트}': 태그 내에 텍스트 입력

예시: h1{유용한 Emmet}

▶ 몇 가지 응용 예시

h1+p*2+script

p#demo{유용한 Emmet을 익히자!}

ul*2>li*3

h1*2+p*3
